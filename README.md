# Computer Science Study

CS 기본기를 주제별로 공부하고 정리하는 스터디 저장소입니다.

첫 번째 스터디는 **OS / OSTEP 한글판**으로 시작합니다. OS는 현재 OSTEP만 기준으로 정리하고, 다른 교재가 필요해지면 나중에 구조를 확장합니다. 이후 네트워크, 컴퓨터 구조, 자료구조, 알고리즘, 데이터베이스 같은 핵심 주제를 순서대로 확장합니다.

## Note Rule

- 저장소 루트는 CS 전체 주제와 진행 현황을 정리합니다.
- 각 주제 디렉토리는 해당 주제의 교재, 진행 방식, 멤버별 정리 링크를 담습니다.
- 개인별 정리는 `주제/이름/chXX/README.md` 형식으로 작성합니다.
- 이미지, 캡처, 다이어그램은 각 챕터의 `images/` 디렉토리에 저장합니다.
- 모르는 부분은 지우지 않고 `헷갈린 점` 또는 `스터디 질문`에 남깁니다.

# Operating System Concepts Study

운영체제 공룡책으로 불리는 **Operating System Concepts**를 함께 읽고 정리하는 스터디 저장소입니다.

## 스터디원

- [곽유섭](./곽유섭/README.md)
- [정주연](./정주연/README.md)

## 정리 방식

각 스터디원은 본인 이름의 디렉토리에서 챕터별로 내용을 정리합니다.

```text
이름/
  README.md
  ch01/
    README.md
    images/
  ch02/
    README.md
    images/
```

- 챕터 정리는 각 챕터 디렉토리의 `README.md`에 작성합니다.
- 그림, 캡처, 다이어그램은 해당 챕터의 `images/` 디렉토리에 저장합니다.
- 이미지 링크는 챕터 README에서 상대 경로로 연결합니다.

```md
![이미지 설명](./images/example.png)
```

## 정리 템플릿

챕터별 README는 아래 흐름을 기준으로 작성합니다.

- 학습 목표
- 핵심 개념
- 주요 내용
- 이미지
- 헷갈린 점
- 스터디 질문
- 참고 자료

## 챕터

현재 각 스터디원 디렉토리에는 `ch01`부터 `ch21`까지 정리 공간이 준비되어 있습니다.
